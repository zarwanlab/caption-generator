# تولیدکننده کپشن اینستاگرام

تولیدکننده کپشن اینستاگرام یک برنامه لاراول تک‌صفحه‌ای برای تولید کپشن محصول، هشتگ و پیشنهاد کوتاه هوشمند برای فروشگاه‌های اینستاگرامی است. رابط کاربری با HTML، CSS و JavaScript خام پیاده‌سازی شده، بنابراین بدون npm و بدون مرحله build فرانت‌اند اجرا می‌شود.

## ویژگی‌ها

- تولید کپشن، هشتگ و پیشنهاد با هوش مصنوعی
- گردش‌کار تک‌صفحه‌ای و انجام تمام عملیات در صفحه اصلی
- پشتیبانی چندزبانه برای انگلیسی، فارسی و عربی
- تغییر زبان همراه با ثبت در پارامتر `lang` داخل URL
- تاریخچه مبتنی بر فایل و ذخیره‌سازی محلی در `storage/app`
- رابط مدرن با الهام از glassmorphism و پشتیبانی کامل RTL
- بدون نیاز به دیتابیس

## پیش‌نیازها

- PHP 8.3 یا بالاتر
- Composer
- یک endpoint سازگار برای chat completion هوش مصنوعی

## نصب

1. وابستگی‌های PHP را نصب کنید:
   ```bash
   composer install
   ```

2. در صورت نیاز فایل محیطی را بسازید:
   ```bash
   cp .env.example .env
   ```

3. کلید برنامه را تولید کنید:
   ```bash
   php artisan key:generate
   ```

4. تنظیمات AI را در فایل `.env` مشخص کنید:
   ```dotenv
   AI_BASE_URL="https://ai.zarwan.co/v1/chat/completions"
   AI_TOKEN="your_token_here"
   AI_MODEL="cf/@cf/openai/gpt-oss-120b"
   ```

5. برنامه را اجرا کنید:
   ```bash
   php artisan serve
   ```

   برنامه در `http://127.0.0.1:8000` در دسترس خواهد بود.

## استفاده

1. برنامه را در مرورگر باز کنید.
2. نوع محصول را انتخاب کنید.
3. زبان کپشن را مشخص کنید.
4. توضیحات محصول را وارد کنید.
5. دکمه تولید را بزنید تا کپشن، هشتگ و پیشنهاد هوشمند دریافت شود.
6. خروجی کامل یا هر بخش را مستقیم از صفحه کپی کنید.

## ذخیره‌سازی

- تاریخچه در فایل JSON محلی `storage/app/history/instagram-caption-generator.json` ذخیره می‌شود.
- هیچ مهاجرت یا seed دیتابیس لازم نیست.
